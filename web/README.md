# DoongD Web Prototype

Vercel-targeted web prototype for DoongD.

## Current scope
- battle-scene-first playable prototype
- wave defense loop
- skill buttons
- auto battle and speed toggle
- result state
- generated draft art integrated into the UI

## Scripts
- `npm run dev`
- `npm run build`
- `npm run start`

## Deployment
Recommended target: Vercel
