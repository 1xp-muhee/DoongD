# DoongD MVP Finalize

> 상태: v0.1
> 목적: 실제 개발 시작용 MVP 확정 초안

---

## 1. MVP에 반드시 포함
- 둥지 1종
- 방 타입 4종 내외 (통로/병목/함정/상업 또는 회복 1종)
- 기본 유닛 6종 내외
- 네임드 2종
- 적 세력 2종
- 기본 방어전 루프
- 소규모 외부 개입 1종
- 기본 승계 시스템
- 1회차 스토리 라인
- 엔딩 2종 내외

### MVP 잠정 고정안
- 방 타입: 통로 / 병목 / 함정방 / 여관방
- 기본 유닛: 탱커 2, 딜러 2, 제어형 1, 지원형 1
- 네임드: 아르벤 로크, 에일린 셰이드 중 2종 우선 사용
- 적 세력: 황야 약탈연맹, 제국 조사단
- 외부 개입: 소형 마을 1종, 계산형 + 기초 브레스 조절만 구현

## 2. MVP에서 제한 포함
- 관계 분기 샘플 2명만
- 비서 시스템은 초기 고정 비서 + 기초 보조만
- 상업 시설은 여관/아이템상 정도로 축소
- 리듬 브레스는 소형 목표 1단계만

## 3. MVP에서 제외/후순위
- 다중 둥지 위치
- 2차/3차 진엔딩
- 마경 자원채취 전체 확장
- 비서 선택 다변화
- 놀이공원/목욕탕 고급 변주
- 진화형 네임드 심화

## 4. 검증 질문
- 둥지 운영/전투/외부 개입이 한 루프로 재밌는가?
- 전투를 보는 맛이 있는가?
- 다음 회차를 하고 싶게 만드는가?
